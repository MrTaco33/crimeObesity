# crimeObesity
Analysis of the still to prove causal relationship between crime and obesity.



Para empezar a trabajar deberás correr los siguientes comandos en tu terminal: 

```
git clone https://github.com/MrTaco33/crimeObesity.git
```
```
cd crimeObesity
```
```
python -m venv env
```
```
source env/bin/activate  # o env\Scripts\activate en Windows 

```
```
#  source env/Scripts/activate en git
```
```
pip install -r requirements.txt
```
